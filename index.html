SCSS Framework

LAMP社内でゼロイチ構築する際に利用するオリジナルフレームワークです。
Tailwind CSSフレームワークの設計思想に影響を受けたマルチクラス＆ユーティリティファーストなコーディングによる柔軟なカスタマイズ性が特徴です。

あらかじめ各cssプロパティに呼応するclassを幅広く用意しておくことで保守や修正作業時にいちいちcssを編集する手間を軽減し、俗人的にならないclass命名規則によって誰でもhtmlを編集するだけでデザインを変更できるようにすることを目的として構築しました。

File Index

styles.scss …全てのscssを梱包しているフォルダを読み込みstyle.cssをコンパイルする

  • global

    • _index.scss

      globalフォルダ内のscssをまとめる

    • _settings.scss

      サイト全体の設定用

    • _mixins.scss

      mixin(使い回しパーツ)管理用

    • _functions.scss

      関数(計算や処理周り)管理用

  • base

    • _index.scss

      baseフォルダ内のscssをまとめる

    • _reset.scss

      ブラウザ間の差異を無くすリセットファイル

    • _base.scss

      html,bodyなどのサイト全体に関わる設定

  • layout

    • _index.scss

      layoutフォルダ内のscssをまとめる

    • _header.scss

      ヘッダーレイアウト構築用

    • _footer.scss

      フッターレイアウト構築用

  • module

    • _index.scss

      moduleフォルダ内のscssをまとめる

    • _project.scss

      ユーティリティクラスで賄えないプロジェクト毎に必要になるパーツを書く

    • _component.scss

      ボタンやタイトルなど汎用的に使い回すコンポーネントとなるパーツを書く

  • utility

    • _index.scss

      utilityフォルダ内のscssをまとめる

    • _layout.scss

      レイアウトに関係するユーティリティ

    • _spacing.scss

      paddingやmarginなどスペーシングに関係するユーティリティ

    • _flexbox.scss

      flexbox周りのユーティリティ

    • _sizing.scss

      widthやheightなどサイズに関係するユーティリティ

    • _borders.scss

      boder周りのユーティリティ

    • _interactivity.scss

      cursorなどインタラクションに関係するユーティリティ

    • _typography.scss

      font-sizeや色などタイポグラフィに関係するユーティリティ

    • _transitions.scss

      transitionアニメーション周りのユーティリティ

    • _animations.scss

      animation keyframeをまとめるユーティリティ

    • _transform.scss

      transform周りのユーティリティ

    • _state.scss

      hoverやPC/SPの切り替えなど状態変化をまとめるユーティリティ＋その他
